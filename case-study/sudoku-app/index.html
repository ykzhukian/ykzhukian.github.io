数独游戏 by React & React Native

2019-10-15 | Case Study

游戏界面

-> 网页地址 Heroku 打开有点慢

规则和功能点

  • 一个简单的 9 x 9 数独游戏
  • 分4个等级的难度，不同难度初始的数字数量不同
  • 单元格不同颜色说明：
    1. 红色：填写的数字有误
    2. 蓝色：双击后标记的单元格
  • 支持保存当前的进度，并且可以回到保存的某个进度，保存的进度支持预览

实现

使⽤ JS 实现数独算法和游戏逻辑。
React 提供的⺴⻚版 和 React Native 提供的⼿机端两个版本。
