Alfred自动生成随机的测试邮箱地址

2022-04-16 | Development

Gmail 邮箱地址后面加上 +xxx@email.com 就可以用原来的邮箱接收测试邮件，避免同一个邮箱重复注册之类的场景。

每次要手动编写比如 kian.zhu+asdfasdf@airwallex.com 这种邮箱，就会很乱，而且也很容易遇到重复。所以，决定用 Alfred 自动生成。去重的目的就用时间戳实现。

1. 创建两个步骤

  • 生成邮箱地址
  • 复制（Alfred 自带的复制到剪切板）

2. 编写邮箱地址输出

  • keyword：email、No Argument
  • 使用 php 语言，拼一个 result

3. 结果展示

  • 唤醒 Alfred 输入 email
